# 项目结构说明
```
.
├── api: 网络请求
├── assets: 静态资源
├── components: 组件
├── icons: 图标
├── international: 国际化
├── layout: 布局
├── mixin: 混入
├── router: 路由
├── store: 状态管理
├── styles: 样式
├── utils: 工具
└── views: 页面
```
# 插件使用说明
+ `ant-design-vue`: UI库
+ `axios`: 网络请求
+ `clipboard`: 复制到剪贴板
+ `echarts`: 图表库
+ `element-plus`: UI库
+ `intro.js`: 引导页
+ `js-cookie`: cookie操作
+ `lodash`: 工具库
+ `pinia`: 状态管理
+ `splitpanes`: 分割面板
+ `vue-count-to`: 数字滚动
+ `vue-i18n`: 国际化
+ `vue-router`: 路由
+ `vuedraggable`: 拖拽
